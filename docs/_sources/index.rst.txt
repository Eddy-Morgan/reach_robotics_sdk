Reach Robotics SDK
=======================================================================

The Reach Robotics SDK contains software tools and interfaces for Reach Robotics Products. 
It enables users to easilty interface and control the different products that Reach Robotics Offers.

The Reach Robotics SDK Support various platforms and programming languages.

Programming Interfaces
-----------------------------

   :width: 25%
   :align: middle


.. |c| image:: images/C_Logo.png

.. |c++| image:: images/ISO_C++_Logo.png
   :width: 25%
   :align: middle


.. centered:: |python|     |c|     |c++|



The Reach Robotics SDK Support Python, C and C++ Programming languages.
We support a range of programming languages. You can interface your product with high level langauges like python.
Also you to easily interface your product with low level systems using the C/C++ SDK even allowing you to easily intergrate with embedded systems.

ROS and ROS 2 Support
-----------------------------

.. figure:: images/Ros_logo.png
   :width: 50%
   :align: center

|

The Reach Robotics SDK Support ROS and ROS2 allowing you to easily and quickly intergrate your product into 
your robotic system. Visualisation Tools and configurations files are included to get you running from the start.


Download
-----------------------------
Download the Reach Robotics SDK from the `Reach Robotics Github. <https://github.com/Reach-Robotics/reach_robotics_sdk>`_
